# WASM Image Kernel Lab

A browser graphics profiling lab that applies convolution filters and compares simulated JavaScript and WASM-style execution paths.

## Highlights
- Kernel presets: sharpen, blur, edge detect, emboss.
- Real-time filter application on procedural textures.
- Runtime and throughput profiling.
- Pixel-difference analysis between execution modes.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS dashboard

## Run
Open `index.html` in your browser.
